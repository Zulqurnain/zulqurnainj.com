# zulqurnainj.com

Portfolio website for Zulqurnain Haider — Senior Full Stack Developer.

Built with Next.js 16, React 19, Tailwind CSS v4, static export deployed to Namecheap shared hosting via GitHub Actions.

## Development

```bash
npm install
npm run dev       # http://localhost:3000
npm run build     # static export to out/
npm run lint
```

## Branches

- `master` — production, triggers auto-deploy to zulqurnainj.com
- `develop` — backup / staging work
